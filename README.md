# Java Fundamentals

A collection of Java fundamentals, core data structures, and coding exercises built as part of my learning journey.

---

## About

This repository contains my implementations of fundamental Java concepts, core data structures, searching and sorting algorithms, and coding exercises. The code is organized topic-wise to make learning, revision, and future reference easier.

The objective of this repository is to strengthen my understanding of Java programming and build a solid foundation before moving on to advanced Data Structures & Algorithms (DSA) and technical interview preparation.
