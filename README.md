# 👋 About

This repository documents my journey of learning **Core Java** through practical coding and hands-on implementations.

It contains Java fundamentals, commonly used data structures, searching and sorting algorithms, and practice problems. The code is organized topic-wise to make learning, revision, and future reference easier.

This repository serves as the foundation for my **Data Structures & Algorithms** learning journey.

# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/Jannat0497/Java-fundamentals.git
```

Navigate to the project

```bash
cd Java-Fundamentals
```

Compile any Java file

```bash
javac FileName.java
```

Run the program

```bash
java FileName
```
